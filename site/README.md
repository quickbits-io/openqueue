# OpenQueue site

Marketing site and documentation for OpenQueue, built with Next.js and
[Fumadocs](https://fumadocs.dev).

```bash
bun install
bun dev
```

- `app/(home)` — landing page
- `app/docs` — documentation, rendered from `content/docs/*.mdx`
- `content/docs` — the docs themselves

`bun run build` produces the production build.
